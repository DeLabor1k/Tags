# Tag & Screen System

## Описание
Система управления тегами и экранами для мониторинга данных в реальном времени с возможностью просмотра графиков и таблиц.

## Технические характеристики

### Tag Service (Сервис тегов)
- **2000 тегов** в системе
- **30 быстрых тегов** - обновление каждые 17 мс
- **1970 обычных тегов** - обновление каждые 50 мс
- Данные хранятся в **JSON файле** (`Database/tags.txt`)
- Автосохранение в БД каждые 5 секунд
- **Queue Size** (1-20) - порог обновлений для прогресс-бара

### Screen Service (Сервис экранов)
- **12 экранов** (фиксированное количество)
- **5 графиков** (экраны 0-4)
- **7 таблиц** (экраны 5-11)
- Каждый экран содержит **300-500 тегов** (случайный выбор)
- Автообновление данных каждые **250 мс**

### Отображение графиков
- Формат вывода: **[X; Y]** где X = время, Y = значение
- Значения Y отображаются в диапазоне **[-2; 0]**
- Каждая точка графика выводится на отдельной строке

### Пользователи
- **Admin** - полный доступ, может редактировать значения тегов
- **Operator** - только просмотр экранов и графиков
